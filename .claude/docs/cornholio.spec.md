# Introduction

Cornholio is a webapp for creating and running a cornhole tournament.

# Content
Cornholio should provide the functionality to set up a Cornhole tournament. First, in an option
screen, you can select the tournament rules. Group stage, round robin, playoff. Do some research
for common tournament setups.

Then the user can enter player names. These can either be put into teams of 2, or shuffled into
teams of 2. If the player number is odd, we can create a 1 player team. 

For the tournament the user enters the final score of a game pairing. It then ends up
in the group table information or show in the elimination bracket.

## Webapp Spec
Suggest some tech stacks for this webapp. It should be kept simple, be responsive, so possible
to use on phone or laptop. For team pairings drag and drop should be supported (if not random).
We don't need a database (for now). Just the option to save the final result. (probably direct html save, but open for better options).

The styling should be fun and entertaining. We can include elements of beavis and butthead "Cornholio" - "I am the greatest." - "Give me TP for my bunghole." in the design. Do some research -
you can use graphical elements from the web if you find some. It will be a private webpage, so no
fear for copyright issues. Use the frontend design skill and playwright mcp for creating the user interface.
